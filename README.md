# Readme
This version extends the original block-only level editor with support for multiple entity types.

## New Features

- **New placeable entities:**
  - Block (square)
  - Enemy (inverted triangle)
  - Catapult (diamond)
  - Bird (circle)
  - Support (vertical rectangle)

- **New buttons in the sidebar** to add each entity type:
  - Add Block, Add Enemy, Add Catapult, Add Bird, Add Support

- **Visual shapes and colors** defined in style.css using:
  - Shared base class: .block
  - Type-specific classes: .type-block, .type-enemy, .type-catapult, .type-bird, .type-support
